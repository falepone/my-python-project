# Python Project Agent Instructions

## 🚀 Fully Automatic GitHub Automation

Bu agent tamamen otomatik şekilde çalışır:

### 1. Otomatik Git Commit & Push (Soruşuz)
- Dosya kaydedildiğinde (`Ctrl+S`), agent tarafından otomatik commit yapılır
- Commit mesajı: "Auto-update: [tarih-saat] - [dosya adı]"
- **Soruşuz olarak** otomatik push yapılır (GitHub'a anında gider)

### 2. Otomatik Pull Request
- Push yapıldığında GitHub Actions otomatik tetiklenir
- Yeni bir PR otomatik olarak açılır
- PR başlığı: "🤖 Auto-update: [timestamp]"

### 3. Otomatik Testler & Merge
- Testler otomatik çalıştırılır (pytest, pylint, black)
- Testler başarılı olursa PR otomatik merge edilir
- Başarısız olursa PR açık kalır (manual review gerekli)

### 2. Kod Yazarken Yardım
- Python kodu yazarken kod suggestions verme
- Hataları tespit etme ve düzeltme önerileri
- Docstring ve type hints eklemek
- Code refactoring önerileri

## Agent Konfigürasyonu

### Aktivasyon Koşulları
- Python dosyaları yazılırken otomatik aktif ol
- Her 30 saniyede bir değişiklikleri kontrol et
- Dosya kaydedilince işleme başla

### Komutlar
- `git add -A` - Tüm değişiklikleri stage et
- `git commit -m "Auto-update: $(date)"` - Otomatik commit
- `git push origin main` - GitHub'a push et

### Hata Yönetimi
- Push başarısızsa 3 kez retry et
- Network hatası varsa uyar ver
- Conflict varsa paused mode'ye geç

## Python Coding Assistant

### Özellikler
1. **Syntax Check**: Her satırı kontrol et
2. **Type Hints**: Türleri otomatik suggest et
3. **Best Practices**: PEP 8 uyumluluğunu kontrol et
4. **Documentation**: Fonksiyonlar için docstring öner
5. **Testing**: Unit test önerileri ver

### Kullanılan Araçlar
- pylance (type checking)
- black (code formatting)
- pylint (linting)
- pytest (testing)

## Not
- Agent yalnızca `my-python-project` klasöründe çalışır
- GitHub token güvenli şekilde saklanır
- Tüm işlemler lokal olarak başlar, sonra push edilir
